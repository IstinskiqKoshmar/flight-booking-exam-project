# SkyRoute - приложение „Самолетни полети“

Индивидуален изпитен проект на **Богомил Иринчев, XI Е** по „Софтуерна разработка по тема“.

## Функционалности

- търсене по начално и крайно летище, дата, брой пътници и вид пътуване;
- филтриране по авиокомпания и директни полети;
- сортиране по цена, продължителност и час на излитане;
- резервация с лични данни и автоматично изчисляване на общата цена;
- управление на резервации и потребителски пътувания по имейл;
- статус на полетите: навреме, закъснял, отменен, излетял и кацнал;
- CRUD операции за `Flight` и `Booking`;
- NUnit тестове, покриващи CRUD за `Booking`.

## Технологии

Приложната логика е написана на **C#**, а данните се съхраняват в **SQL** база SQLite. Използвани са ASP.NET Core MVC, Entity Framework Core Code First и NUnit. Razor и CSS служат единствено за визуалното представяне; няма JavaScript логика.

## Стартиране

```text
dotnet restore Bogomil_Irinchev_XI_E.sln
dotnet run --project src/FlightBooking.Web
```

При първото стартиране базата се създава автоматично в `src/FlightBooking.Web/App_Data/flightbooking.db` и се попълва с примерни данни.

## Структура

- `FlightBooking.Core` - модели на данните;
- `FlightBooking.Data` - DbContext, Code First конфигурация, seed данни и repository слой;
- `FlightBooking.Web` - MVC контролери и изгледи;
- `FlightBooking.Tests` - автоматизирани NUnit тестове;
- `Database/schema.sql` - SQL схема за справка.
